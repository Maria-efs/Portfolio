Meus Projetos

Coca-Cola

Esse foi o primeiro projeto desenvolvido em aula junto com o professor, essa página montamos com ajuda total do professor e aprendemos conteúdos básicos para montar uma página simples com poucos elementos tipo imagens, textos e cores. Após esse projeto fizemos páginas mais complexas. Foi feito o uso de HTML, CSS e do Figma.

Personagem

Esse foi o segundo projeto desenvolvido em aula e foi o primeiro projeto onde realizamos a página sozinhos. O professor pediu uma página sobre um personagem qualquer e com alguns elementos, produzimos ela utilizando e treinando os conteúdos aprendidos em aula e desenvolvendo mais os conceitos de header, main e footer. Usamos HTML, CSS e o Figma.

Doctor Strange

Esse foi o segundo projeto que desenvolvemos juntos com o professor, mas o diferencial dele foi que era uma página mais complexas, ao logo do desenvolvimento da página fomos aprendendo conteúdos diferentes e esse foi o nosso primeiro contato com a versão mobile e a responsividade. A página inicial foi feita junto com o professor mais o restante foi produzido pelos alunos e cada um fez de um jeito. Usamos HTML, CSS, Figma e JavaScript.

Projeto Integrador

O principal diferencial desse projeto foi que esse site foi um trabalho integrado com os alunos do curso de ADM, nesse momento tivemos a experiência de ter o contato com um cliente, onde os alunos de ADM eram nossos clientes e buscaram a gente para desenvolver um site para a empresa que eles criaram. Tivemos, não só o aprendizado de como tratar o cliente, mas também o desenvolvimento de novos conteúdos para desenvolver o site e essa foi a página que eu fiz. Usamos HTML, CSS, Figma e JavaScript.
